AquariumThermostat
==================

Firmware of an aquarium thermostat which is constructed with a H8-3694 micro controller. 
This firmware supports command base serial communication with pc, thermo controll, and light on/off.

Build Environment
------

This source repository contains a project of HEW(High-performance Embedded Workshop).
Please install it and build source code.
